   Die I/O Platine der ATW800

   J1            = Netzteilanschluß
   J2            = Romportbuchse
   JI01          = Midi Buchse
   J5            = Tastaturanschluß
   JI01          = Centronicsbuchse
   JI01          = Modem 1 (MFP-Chip)
   J9            = Batterieanschluß für Echtzeituhr
   J10           = ACSI DMA Anschluß für externe Geräte
   J12           = Stiftleiste für Anschluß der internen Floppy
   J13           = Anschluß für Floppy B (externes Laufwerk)
   J14           = Monitorbuchse
   J15           = MegaBus Stiftleiste
   J17           = ACSI DMA Anschluß für interne Geräte
   J18           = ????

   J202          = ????
   J205          = Verbindung zur Frontplatine (Statusled und Resettaster)
   S1            = Resettaster

   W1/W5         = Driveselect Jumper

   LEDC          = ????
   JP3           = ????
   JP20          = ????




   Die Hauptplatine der ATW800

   J1 bis J4     = Transputerlinks

   J10 bis J13   = Steckplätze für Farmcards

   J22           = Netzteilanschluß
   J23           = Spannungsversorgung für I/O Platine

   SM1 bis SM4   = Bestückungsplätze für SIPP (SIMM) Module


   Die Frontplatine der ATW800

   J100          = Verbindung zur Platine (Link aktiv?, etc. (E,A,P))
   J101          = Anschluß des Resettaster
   J102          = Anschluß der Harddisk LED


   Die Festplatten Treibersoftware für die ATW800

   SCSICAP .      1962 Bytes
   M4IDMA  .PRG  20745 Bytes
   MSCSIFMT.PRG  43412 Bytes
   SCSIFMT .RSC   4730 Bytes

   Diese funktioniert NUR in verbindung einer ATW800, das starten auf
   einer anderen Maschine wird mit einer Fehlermeldung bzw. Bomben
   quittiert!
