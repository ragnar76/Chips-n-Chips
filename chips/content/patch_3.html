   STE TOS, Rev 2 (TOS 1.62)
   --------------

   This is a slightly revised revision of TOS 1.6. It fixes the POOL-
   FIX problem in GEMDOS and the problem in the Desktop that was pre-
   sent in TOS 1.06.

        Formats:   2 chip ROMs (or 1MB Eprom DIL-Chip)

       Machines:   520STE, 1040STE

      Utilities:   CACHEXXX.PRG, FOLDRXXX.PRG

                   CACHEXXX.PRG und FOLDRXXX.PRG are explained above.

          Notes:   This version is only available in 1040STE and 520
                   STE machines. It isn't available for upgrading.

        Formats:  2 chip ROMs (a 1MB Eprom DIL-Chip)

       Machines:  Mega STE

      Utilities:  CACHEXXX.PRG, FOLDRXXX.PRG

                  CACHEXXX.PRG und FOLDRXXX.PRG are explained above.

          Notes:  This version is only available in the Mega STE.  It
                  isn't available for upgrading.



   Mega STE TOS (TOS 2.06)
   ------------

   TOS 2.06 is the latest version of TOS currently shipping in the
   Mega STE und STBook, fixed from ATARI for TOS-Upgrade with a TOS-
   Card for all ST's

   Special Version with other Date and fixed Routines for STBook, this
   version is not compatible with the Version for ST/STE Computers!

        Formats:  2 chip ROMs (a 1MB Eprom DIL-Chip)
                  1 chip ROM (a PLCC Chip) in a STBook

       Machines:  Mega STE, STBook

      Utilities:  CACHEXXX.PRG, FOLDRXXX.PRG

                  CACHEXXX.PRG und FOLDRXXX.PRG are explained above.

          Notes:  This version is available on ST/MegaST/MegaSTE. It
                  is for upgrade for all older TOS versions on
                  ST-Computers.
