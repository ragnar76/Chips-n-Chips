   Idle (hh:mm:ss)        00:01:00

   Hier kann ein Wert eingegeben werden, nach dessen Zeit sich die
   Festplatte selbst abschaltet, wenn kein Zugriff mehr getätigt wurde.

   Offiziell ist dieses Feature nur in der Version 3.02 (STACY Laptop)
   und 5.00 (STBook Notebook) vorhanden!


   Scan physical units    ( ) All ACSI

                          ( ) Custom ACSI

                          ( ) All SCSI

                          ( ) Custom SCSI

                          ( ) IDE Unit 0

   Hier kann eingestellt werden, ob alle Geräteadressen (ALL) oder nur
   spezielle Geräteadressen (CUSTOM) beim Systemstart abgefragt werden
   sollen.

   IDE Unit 0 ist nur beim STBook, Falcon030 und Computern mit TOS 2.06
   AT-Bus/IDE-Interface wirksam!


   Removable Media Drive Letters

        Unit     ACSI       SCSI

       0 - 7     0-14       0-14

   Hier können die Anzahl der logischen Laufwerke/Partitionen einge-
   stellt werden, die beim Systemstart automatisch angemeldet werden
   sollten, wenn eine Wechselplatte erkannt wird.




   Achtung:

   Bei vorhandener IDE-Hardware (STBook, Falcon030, AT-BUS TOS 2.06
   Karte) legt das TOS nach dem überprüfen der angeschlossenen Disket-
   tenlaufwerke eine Pause von 5 Sekunden ein, falls sich die IDE-
   Festplatte nicht sofort meldet!

   Startet dann der Harddisktreiber und fragt aufgrund seiner Konfi-
   guration nochmal den IDE-Port ab, entsteht wieder eine Wartepause!

   Dem ATARI Harddisktreiber, kann man diese Zeit abgewöhnen wenn man
   innerhalb von AHDI.PRG / SHDRIVER.SYS AB Version 5.00 nach folgen-
   dem String sucht:

   nach  $00 $00 $03 $E8 suchen
   durch $00 $00 $00 $00 ersetzen
