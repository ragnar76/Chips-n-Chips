   RS232 Fehlercode: (Teststecker verwenden!)

   S0= Data not received. Überprüfe Signalweg: MFP pin 8 nach J6 pin 2
       via 1488 to J6 pin 3 nach MFP pin 9 via 1489.

   S1= Data mismatch. Siehe Fehler S0.

   S2= Input frame error. MFP auswechseln.

   S3= Input parity error. MFP auswechseln.

   S4= Input data overrun. Es werden Daten schneller ausgegeben als die
       CPU lesen kann. MFP oder GLUE defekt.

   S5= No IRQ. CPU bekommt keinen Interrupt wenn ein Signal empfangen
       wird. MFP oder GLUE defekt.

   S6= Transmit error. MFP auswechseln.

   S7= Transmit error interrupt. MFP auswechseln.

   S8= Receive error interrupt. MFP auswechseln.

   S9= RI/DTR connection. RI hat mit DTR keine Verbindung.

   SA= DCD/DTR connection. DTR hat keine Verbindung mit DCD.

   SB= RTS/CTS connection. RTS hat keine Verbindung mit CTS.

   Teststecker für RS232 Test: Es müssen folgende Pin's verbunden
   werden: Pin 2 und 3,Pin 4 und 5, Pin 8, 20 und 22


   DMA Fehlercode: (Platine anschliessen!)

   D0= DMA time out. Folgende Pegel überprüfen HDINT und HDRQ (Pin 23).
       Vergleiche die Signale HDINT mit INTR Pin 34 am MFP.

   D1= DMA counter error. MMU oder DMA defekt.

   D2= Data mismatch error. DMA auswechseln, wenn dann noch Fehler die
       Datenleitungen auf Unterbrechung oder Kurzschluß vom Chip zum
       Port überprüfen. Eventuell auch defekter Floppy Controller siehe
       Floppy test.

   Druckerport und Joystickport Fehlercode: (Platine anschliessen!)

   P0= Printer port error.

   P1= Busy Signal failed.

   J0= Joystick port 0 failed.

   J1= Joystick port 1 failed.

   J2= Keyboard prozessor timed-out.

   J3= Left button line failed. Linke Maustaste defekt.

   J4= Right button line failed. Rechte Maustaste defekt.
