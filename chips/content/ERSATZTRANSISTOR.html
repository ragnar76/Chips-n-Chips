   Der Transistortyp 2N3904 wird von ATARI als Standardtransistor
   überall eingebaut (Soundchip, Treibertransitoren der Videosignale,
   etc). Als Europäischen Ersatztyp eignet sich z.B. der BC546B, al-
   lerdings sind dort Emitter und Collector vertauscht (die beiden
   äusseren Beinchen).

   2N3904 von unten:    -----------------
                        | C     B     E |
                        |               |
                         \             /
                          \           /
                           -----------

   BC546B von unten:    -----------------
                        | E     B     C |
                        |               |
                         \             /
                          \           /
                           -----------

   Falls mal garkein Austauschexemplar aufzutreiben sein sollte, gibt
   es nach "Edgar Fuß" noch einen Algorithmus der helfen könnte:


    IF Unbekannt(Transistor) THEN BEGIN
      IF Transistor[2] IN ['0'..'9'] THEN CASE Transistor[1] OF
         'A','B','J','K': BEGIN
           Insert('2S',Transistor,1);
           Lookup(Transistor,Japan);
         END;
         'F': BEGIN
           Insert('B',Transistor,1);
           Lookup(Transistor,Europa);
         END;
         'C','D': BEGIN
           IF HatSchlitzaugen(Transistor) THEN BEGIN
             Insert('2S',Transistor,1);
             Lookup(Transistor,Japan);
           END ELSE BEGIN
             Insert('B',Transistor,1);
             Lookup(Transistor,Europa);
           END;
         END;
         ELSE Fail;
      END;
    END;


   Die Bezeichnungen europäischer Transistoren beginnen fast immer mit
   "BC" oder "BD", gefolgt von einer meist dreistelligen Zahl und einem
   optionalen Kennbuchstaben für die Stromverstärkung (A, B oder C).
   Gelegentlich wird das führende "B" beim Aufdruck auf dem Bauteil
   weggelassen. Die Bezeichnungen für japanische Standard-Transistoren
   beginnen immer mit "2S", gefolgt von einem der Buchstaben 'A','B',
   'C','D','J','K' und einer meist vierstelligen Zahl. Sehr häufig wird
   das Präfix "2S" beim Aufdruck der Bauteile weggelassen. Nur bei den
   Buchstaben 'C' oder 'D' kann dies zu Zweideutigkeiten mit europä-
   ischen Transistoren führen, die allerdings so gut wie nie vierstel-
   lige Zahlen benutzen. Notfalls muß man beide Typen versuchen, in
   Tabellen zu finden. Amerikanische Transistoren werden mit "2N", ge-
   folgt von einer meist vierstelligen Zahl bezeichnet.
