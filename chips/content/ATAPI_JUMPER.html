  Die Jumperleiste der ATAPI Laufwerke                                   


  C S M     C S M     C S M          T T P/
  S L A     S L A     S L A          0 1 A
  o o o     o o o     o o o          o o o
  |     or      |  or   |                |
  o o o     o o o     o o o          o o o

   (A)       (B)       (C)            (D)

(A) CS = CSEL = Cable Select
     The device shall use the host interface
     signal CSEL to configure the device
(B) MA = Master
     The device shall use the master
(C) SL = Slave
     The device shall use the slave
(D) T0, T1 are used as drive test mode
     P/A = Prevent/Allow
     This jumper enables/disables eject function
