   Der ACB-4000A MFM-zu-SCSI-Controller von Adaptec

   Note: This board does not connect directly to the host's bus.
         Instead, it allows a SCSI host adapter to control up to 2 MFM
         drives. The SCSI host adapter connects to this board, which
         in turn is connected to up to 2 MFM drives.


   Drives:        2 MFM drives (see note above).

   J0             Drive 0 data connector.
   J1             Drive 1 data connector.
   J2             HD control connector.
   J3             DC power connector.
   J4             Internal SCSI connector (to SCSI host adapter).

   P - K          Pulse select protocol used by SASI standard.
   K - L          (Default)  Level select protocol use by SCSI standard.

   PU - R         Precomp disabled for both drives.
   R  - S         Precomp begins at RWC cylinder for both drives.
   R  - T         Precomp enabled on all tracks for both drives.


   J5 -1 Open     SCSI ID 0.  Closed   SCSI ID 1.  Open     SCSI ID 2.
      -2 Open                 Open                 Closed
      -3 Open                 Open                 Open

         Closed   SCSI ID 3.  Open     SCSI ID 4.  Closed   SCSI ID 5.
         Closed               Open                 Open
         Open                 Closed               Closed

         Open     SCSI ID 6.  Closed   (Default)  SCSI ID 7.
         Closed               Closed
         Closed               Closed

      -4 Open     DMA xfer rate = DATACLOCK / 2.
         Closed   DMA xfer rate = SYSCLOCK  / 4.

      -5 Closed   Extended command set enabled (should be used only
                  when replacing SASI controllers).
         Open     (Default)  Extended command set disabled.

      -6          Reserved.

      -7 Closed   Support for drives that drop the SEEK COMPLETE signal
                  when switching heads (ie. Syquest 312 and DMA 360).
         Open     (Default)  SEEK COMPLETE supported not necessary.

      -8 Open     Default - Diagnostic mode disabled (normal operation)
         Closed   Manufacturer diagnostic mode enabled.


   Terminators:   Default is installed. If this is not the 1st or last
                  in a series of SCSI devices, or if inline terminators
                  are in use, remove both terminating resistor packs.
