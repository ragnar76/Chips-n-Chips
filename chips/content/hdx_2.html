   Die Treibersoftware für die ATW800

   SCSICAP .      1962 Bytes
   M4IDMA  .PRG  20745 Bytes
   MSCSIFMT.PRG  43412 Bytes
   SCSIFMT .RSC   4730 Bytes

   Diese funktioniert NUR in verbindung einer ATW800, das starten auf
   einer anderen Maschine wird mit einer Fehlermeldung bzw. Bomben
   quittiert!




   Mit dem Zusatzprogramm HDPATCH.PRG kann der SHDRIVER.SYS/AHDI.PRG

                          AB VERSION 3.00

   nach eigenen Wünschen konfiguriert werden.


   OS Pool Size           128 (default)

   Die Anzahl der Blöcke wird zu denen des ROM TOS addiert.


   Default Sector Size    512 - 16384

   Hier sollte die Größe so angepasst werden, das z.B. bei einer Wech-
   selplatte alle Partitionen beim Medienwechsel erkannt werden, ohne
   daß ein Reset ausgelöst werden muß.
