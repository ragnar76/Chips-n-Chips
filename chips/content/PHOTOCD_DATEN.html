  Die Auflösungen der Photo CD Bilder                                    


Die Auflösungen nennen sich

/16 BASE   192  x  128 Pixel in TC, unkomprimiert,  64kB
/ 4 BASE   384  x  256 Pixel in TC, unkomprimiert, 256kB
     BASE   768  x  512 Pixel in TC, unkomprimiert, ca. 1MB
x 4 BASE  1536  x 1024 Pixel in TC, komprimiert, ca.  5 MB
x16 BASE  3072  x 2048 Pixel in TC, komprimiert, ca. 18 MB


auf der PRO PhotoCD noch zusätzlich:

x64 BASE  6144  x 4096 Pixel in TC, kompimiert, ca. 72 MB
